Cryptogram Game

How to Play Cryptogram

Objective: Decode the encrypted quote by figuring out which letters correspond to which.

Time Limit: You have 5 minutes to solve each cryptogram puzzle. A timer will start as soon as the puzzle appears, and if the time runs out, the answer will be revealed automatically.

Hint Limit: You can use up to 2 hints per puzzle.

Steps to Play:

  1. Each letter in the quote is replaced by a different letter using a Caesar cipher.
  2. Type your guesses in the boxes.
  3. If you're unsure, click the "Hint" button. First, select the cell where you want the hint to appear by clicking it or moving your cursor there.
  4. Once you’ve entered your guesses, click "Check It!" to see if your answer matches the original quote.
  5. If you’d like to try decoding a different quote, click the "New Quote" button. This will replace the current encrypted quote with a fresh one, allowing you to start a new puzzle.

Tips for Decoding:

  • Identify short words: Words like "THE," "AND," and "IS" are often present. Guessing these can give you clues to other letters.
  • In a Caesar cipher, each letter shifts by the same number. Try shifting all letters forward or backward to see if words start to make sense!
