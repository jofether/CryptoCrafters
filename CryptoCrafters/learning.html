Cryptography

Cryptography is the practice and study of techniques for securing communication and information. It ensures the confidentiality, integrity, and authenticity of data.

How Cryptography Works?

Cryptography works by transforming information into a secure format that is unreadable without a specific key. This transformation ensures that even if data is intercepted, it remains confidential.

The Caesar Cipher is a substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet.

Cryptography Vs. Cryptanalysis

Cryptography

Cryptography is the art of creating secure communication, ensuring that data is protected from unauthorized access.

Cryptanalysis

Cryptanalysis is the art of breaking secure communications to understand the encrypted data without the key.

Terms in Cryptography

Plaintext

original readable message or data.

Example: "Hello World"

Ciphertext

encrypted output that is not readable without decryption.

Example: "U2FsdGVkX19/2sJ7a7Z4Nw=="

Encryption

process of converting plaintext into ciphertext using an algorithm and a key.

Example: AES encryption.

Decryption

process of converting ciphertext back into plaintext using a key.

Example: Using the same AES key to get back the original message.

Types of Cryptography

Symmetric-Key Cryptography

Uses the same key for both encryption and decryption.

Asymmetric-Key Cryptography

Uses a pair of keys, one for encryption and a different one for decryption.

Hash Functions

Converts input data of any size to a fixed-size string, which is typically a hash value.

Stream Ciphers

Encrypts data one bit or byte at a time.

Block Ciphers

Encrypts data in fixed-size blocks, typically 64 or 128 bits at a time.

Application of Cryptography

Cryptography is widely used in various applications such as securing online transactions, protecting sensitive information, and ensuring secure communications over the internet.

Goals of Cryptography

The primary goals of cryptography include ensuring confidentiality, integrity, authentication, and non-repudiation of information.
